---
title: OpenAI Chat Completions
description: Connect OpenUI to a backend using OpenAI's chat completions streaming format.
---

Use `openAIAdapter` when your backend streams responses using `openai.chat.completions.create({ stream: true })`.

If your backend also stores and loads messages in OpenAI's `{ role, content }` format, add `openAIMessageFormat` as well.

### Stream only

If you only need to parse the streaming response (messages are stored in OpenUI's default format):

### With OpenAI message format

If your backend also stores messages in OpenAI's `{ role, content }` format and your thread endpoints return them in that shape:

**`openAIAdapter()`** parses the streaming response from OpenAI's chat completions API. The stream consists of newline-delimited JSON chunks, each containing a `delta` with the next piece of content. The adapter reads these chunks and converts them into OpenUI's internal stream events.

**`openAIMessageFormat`** is only needed if your backend stores and loads messages in OpenAI's `{ role, content }` format. It converts between this format and OpenUI's internal format when sending chat requests and loading saved threads.
