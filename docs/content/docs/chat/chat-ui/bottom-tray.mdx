The BottomTray component adds a collapsible chat widget that floats at the bottom of your page — similar to the support chat widgets you see on many websites.

## Basic usage

```tsx
import { BottomTray } from "@openuidev/react-ui";

function App() {
  return (
    <>
      <main>{/* Your app content */}</main>
      <BottomTray apiUrl="/api/chat" agentName="Assistant" />
    </>
  );
}
```

By default, BottomTray starts collapsed. The user clicks to expand it.

## Controlling open state

You can control whether the tray is open or closed from your app:

```tsx
import { BottomTray } from "@openuidev/react-ui";
import { useState } from "react";

function App() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <main>
        <button onClick={() => setIsOpen(true)}>Chat with us</button>
      </main>
      <BottomTray
        apiUrl="/api/chat"
        agentName="Assistant"
        isOpen={isOpen}
        onOpenChange={setIsOpen}
      />
    </>
  );
}
```

Or set it to open by default:

```tsx
<BottomTray apiUrl="/api/chat" agentName="Assistant" defaultOpen />
```

## Full example

```tsx
import { BottomTray } from "@openuidev/react-ui";

function App() {
  return (
    <>
      <main>{/* App content */}</main>
      <BottomTray
        apiUrl="/api/chat"
        threadApiUrl="/api/threads"
        agentName="Support"
        logoUrl="/logo.png"
        defaultOpen
        welcomeMessage={{
          heading: "Need help?",
          body: "Our AI assistant is here to help.",
        }}
        conversationStarters={[
          { label: "Track order", message: "Where is my order?" },
          { label: "Returns", message: "How do I return an item?" },
        ]}
      />
    </>
  );
}
```

## Props

BottomTray accepts all shared chat config and UI props, plus these layout-specific props:

### BottomTray-specific props

| Prop           | Type                        | Default | Description                                   |
| -------------- | --------------------------- | ------- | --------------------------------------------- |
| `isOpen`       | `boolean`                   | —       | Controlled open state                         |
| `onOpenChange` | `(isOpen: boolean) => void` | —       | Called when the user opens or closes the tray |
| `defaultOpen`  | `boolean`                   | `false` | Whether the tray starts open (uncontrolled)   |

### Shared props

See [Overview](/docs/chat/chat-ui) for the full list.

| Prop                   | Type                         | Description                                                     |
| ---------------------- | ---------------------------- | --------------------------------------------------------------- |
| `apiUrl`               | `string`                     | Endpoint that receives messages and returns a streamed response |
| `threadApiUrl`         | `string`                     | Base URL for thread persistence endpoints                       |
| `agentName`            | `string`                     | Name shown in the chat header                                   |
| `logoUrl`              | `string`                     | Logo image shown in the chat header                             |
| `welcomeMessage`       | `WelcomeMessageConfig`       | Heading and body for the empty-chat screen                      |
| `conversationStarters` | `ConversationStartersConfig` | Suggested prompts shown on the welcome screen                   |
| `assistantMessage`     | `AssistantMessageComponent`  | Custom renderer for assistant messages                          |
| `userMessage`          | `UserMessageComponent`       | Custom renderer for user messages                               |
| `processMessage`       | `function`                   | Custom fetch function (use instead of `apiUrl`)                 |
| `streamProtocol`       | `StreamProtocolAdapter`      | Parser for the backend's streaming format                       |
| `messageFormat`        | `MessageFormat`              | Converter for the backend's message format                      |
