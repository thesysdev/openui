---
title: Conversation History
description: Persist chat threads so users can return to previous conversations.
---

By default, conversations live in memory and are lost when the page reloads. To persist them, you can either point to a REST API or provide your own functions.

## Using `threadApiUrl`

The simplest approach — pass a base URL and OpenUI handles the rest:

<Tabs groupId="layout" items={['Copilot', 'FullScreen', 'BottomTray']} persist>
  <Tab value="Copilot">
    ```tsx
    import { Copilot } from "@openuidev/react-ui";

    function App() {
      return (
        <Copilot
          apiUrl="/api/chat"
          threadApiUrl="/api/threads"
          agentName="My Agent"
        />
      );
    }
    ```

  </Tab>

OpenUI expects the following endpoints at your `threadApiUrl`:

| Action        | Method   | URL                         | Body               | Response                                  |
| ------------- | -------- | --------------------------- | ------------------ | ----------------------------------------- |
| List threads  | `GET`    | `/api/threads`              | —                  | `{ threads: Thread[], nextCursor?: any }` |
| Create thread | `POST`   | `/api/threads`              | `{ firstMessage }` | `Thread`                                  |
| Delete thread | `DELETE` | `/api/threads/:id`          | —                  | —                                         |
| Update thread | `PUT`    | `/api/threads/:id`          | `Thread`           | `Thread`                                  |
| Load messages | `GET`    | `/api/threads/:id/messages` | —                  | `Message[]`                               |

A `Thread` object looks like:

```typescript
type Thread = {
  id: string;
  title: string;
  createdAt: string | number;
};
```

See [Thread API Endpoint](/docs/chat/backend/thread-api-endpoint) for how to build this on the server.

## Custom thread functions

If your API doesn't follow the REST pattern above — for example, you need auth headers, a different URL structure, or want to call a GraphQL API — you can pass individual functions:

<Tabs groupId="layout" items={['Copilot', 'FullScreen', 'BottomTray']} persist>
  <Tab value="Copilot">
    ```tsx
    import { Copilot } from "@openuidev/react-ui";

    function App() {
      return (
        <Copilot
          apiUrl="/api/chat"
          fetchThreadList={async () => {
            const res = await fetch("/my-api/conversations", {
              headers: { Authorization: `Bearer ${getToken()}` },
            });
            return res.json(); // { threads, nextCursor? }
          }}
          createThread={async (firstMessage) => {
            const res = await fetch("/my-api/conversations", {
              method: "POST",
              headers: { Authorization: `Bearer ${getToken()}` },
              body: JSON.stringify({ message: firstMessage.content }),
            });
            return res.json(); // Thread
          }}
          deleteThread={async (id) => {
            await fetch(`/my-api/conversations/${id}`, {
              method: "DELETE",
              headers: { Authorization: `Bearer ${getToken()}` },
            });
          }}
          loadThread={async (threadId) => {
            const res = await fetch(`/my-api/conversations/${threadId}/messages`, {
              headers: { Authorization: `Bearer ${getToken()}` },
            });
            return res.json(); // Message[]
          }}
          agentName="My Agent"
        />
      );
    }
    ```

  </Tab>

### Function reference

| Function          | Signature                                                  | Purpose                                                           |
| ----------------- | ---------------------------------------------------------- | ----------------------------------------------------------------- |
| `fetchThreadList` | `(cursor?) => Promise<{ threads: Thread[]; nextCursor? }>` | Load the list of conversations (supports pagination via cursor)   |
| `createThread`    | `(firstMessage: UserMessage) => Promise<Thread>`           | Create a new conversation when the user sends their first message |
| `deleteThread`    | `(id: string) => Promise<void>`                            | Delete a conversation                                             |
| `updateThread`    | `(thread: Thread) => Promise<Thread>`                      | Update a conversation (e.g. rename)                               |
| `loadThread`      | `(threadId: string) => Promise<Message[]>`                 | Load all messages for a conversation                              |
