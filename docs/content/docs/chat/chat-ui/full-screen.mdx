The FullScreen component renders a complete chat interface that takes up the entire page — including a sidebar for conversation history, a message area, and a composer.

## Basic usage

```tsx
import { FullScreen } from "@openuidev/react-ui";

function App() {
  return <FullScreen apiUrl="/api/chat" agentName="My Agent" logoUrl="/logo.png" />;
}
```

FullScreen fills its parent container. Make sure the parent has a defined height (e.g. `height: 100vh`).

## Full example

```tsx
import { FullScreen } from "@openuidev/react-ui";

function App() {
  return (
    <FullScreen
      apiUrl="/api/chat"
      threadApiUrl="/api/threads"
      agentName="My Agent"
      logoUrl="/logo.png"
      welcomeMessage={{
        heading: "Welcome!",
        body: "Start a conversation or pick up where you left off.",
      }}
      conversationStarters={[
        { label: "Summarize my data", message: "Can you summarize my latest data?" },
        { label: "Create a report", message: "Help me create a report." },
      ]}
    />
  );
}
```

## Props

FullScreen accepts all shared chat config and UI props. See [Overview](/docs/chat/chat-ui) for the full list.

| Prop                   | Type                         | Description                                                     |
| ---------------------- | ---------------------------- | --------------------------------------------------------------- |
| `apiUrl`               | `string`                     | Endpoint that receives messages and returns a streamed response |
| `threadApiUrl`         | `string`                     | Base URL for thread persistence endpoints                       |
| `agentName`            | `string`                     | Name shown in the chat header                                   |
| `logoUrl`              | `string`                     | Logo image shown in the chat header                             |
| `welcomeMessage`       | `WelcomeMessageConfig`       | Heading and body for the empty-chat screen                      |
| `conversationStarters` | `ConversationStartersConfig` | Suggested prompts shown on the welcome screen                   |
| `assistantMessage`     | `AssistantMessageComponent`  | Custom renderer for assistant messages                          |
| `userMessage`          | `UserMessageComponent`       | Custom renderer for user messages                               |
| `processMessage`       | `function`                   | Custom fetch function (use instead of `apiUrl`)                 |
| `streamProtocol`       | `StreamProtocolAdapter`      | Parser for the backend's streaming format                       |
| `messageFormat`        | `MessageFormat`              | Converter for the backend's message format                      |
