title: Welcome & Starters
description: Configure the welcome experience and conversation starter prompts.
---

When a user opens your chat with no messages, OpenUI shows a welcome screen. You can customize the welcome message and provide suggested prompts (conversation starters) to help users get started.

## Welcome message

Use the `welcomeMessage` prop to set a heading and body text for the empty-chat screen:

<Tabs groupId="layout" items={['Copilot', 'FullScreen', 'BottomTray']} persist>

    ```tsx
    import { Copilot } from "@openuidev/react-ui";

    function App() {
      return (
        <Copilot
          apiUrl="/api/chat"
          agentName="My Agent"
          welcomeMessage={{
            title: "Hi there! 👋",
            description: "I'm your AI assistant. Ask me anything about your account, orders, or products.",
          }}
        />
      );
    }
    ```

### `WelcomeMessageConfig`

Can be either a custom React component or a config object:

```typescript
type WelcomeMessageConfig =
  | React.ComponentType<any>
  | {
      title?: string;
      description?: string;
      image?: { url: string } | ReactNode;
    };
```

## Conversation starters

Conversation starters are clickable prompts shown below the welcome message. When a user clicks one, it sends that prompt as a message automatically:

<Tabs groupId="layout" items={['Copilot', 'FullScreen', 'BottomTray']} persist>

    ```tsx
          welcomeMessage={{
            title: "Hi there! 👋",
    ```

### `ConversationStartersConfig`

```typescript
interface ConversationStartersConfig {
  variant?: "short" | "long"; // "short" for pill buttons (default), "long" for list items
  options: Array<{
    displayText: string; // Text shown on the button
    prompt: string; // Message sent when clicked
    icon?: ReactNode; // Optional icon
  }>;
}
```

The `displayText` is what the user sees on the button. The `prompt` is what actually gets sent to the chat. This lets you show a short, friendly label while sending a more detailed prompt to the LLM.
