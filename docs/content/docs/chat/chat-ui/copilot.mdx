The Copilot component adds a chat panel that sits beside your main application content, like a sidebar assistant.

## Basic usage

```tsx
import { Copilot } from "@openuidev/react-ui";

function App() {
  return (
    <div style={{ display: "flex", height: "100vh" }}>
      <main style={{ flex: 1 }}>{/* Your app content */}</main>
      <Copilot apiUrl="/api/chat" agentName="My Agent" logoUrl="/logo.png" />
    </div>
  );
}
```

The Copilot panel renders alongside your content. It doesn't manage layout of your app — you control where it sits using standard CSS (e.g. flexbox as shown above).

## Full example

```tsx
import { Copilot } from "@openuidev/react-ui";

function App() {
  return (
    <div style={{ display: "flex", height: "100vh" }}>
      <main style={{ flex: 1 }}>{/* App content */}</main>
      <Copilot
        apiUrl="/api/chat"
        threadApiUrl="/api/threads"
        agentName="My Agent"
        logoUrl="/logo.png"
        welcomeMessage={{
          heading: "Hi there! 👋",
          body: "I can help you with anything in this app.",
        }}
        conversationStarters={[
          { label: "Get started", message: "How do I get started?" },
          { label: "Help", message: "What can you help me with?" },
        ]}
      />
    </div>
  );
}
```

## Props

Copilot accepts all shared chat config and UI props. See [Overview](/docs/chat/chat-ui) for the full list.

| Prop                   | Type                         | Description                                                     |
| ---------------------- | ---------------------------- | --------------------------------------------------------------- |
| `apiUrl`               | `string`                     | Endpoint that receives messages and returns a streamed response |
| `threadApiUrl`         | `string`                     | Base URL for thread persistence endpoints                       |
| `agentName`            | `string`                     | Name shown in the chat header                                   |
| `logoUrl`              | `string`                     | Logo image shown in the chat header                             |
| `welcomeMessage`       | `WelcomeMessageConfig`       | Heading and body for the empty-chat screen                      |
| `conversationStarters` | `ConversationStartersConfig` | Suggested prompts shown on the welcome screen                   |
| `assistantMessage`     | `AssistantMessageComponent`  | Custom renderer for assistant messages                          |
| `userMessage`          | `UserMessageComponent`       | Custom renderer for user messages                               |
| `processMessage`       | `function`                   | Custom fetch function (use instead of `apiUrl`)                 |
| `streamProtocol`       | `StreamProtocolAdapter`      | Parser for the backend's streaming format                       |
| `messageFormat`        | `MessageFormat`              | Converter for the backend's message format                      |
