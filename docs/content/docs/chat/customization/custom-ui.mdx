If the built-in chat components (`Copilot`, `FullScreen`, `BottomTray`) don't fit your needs, you can build your own chat UI from scratch using `ChatProvider` and hooks from `@openuidev/react-headless`. You bring the components — OpenUI handles the state, streaming, and thread management.

## `ChatProvider`

Wrap your UI in `ChatProvider` to set up the chat backend. It accepts the same config props as the built-in chat components — `apiUrl`, `threadApiUrl`, `processMessage`, `streamProtocol`, `messageFormat`, and all the custom thread functions.

```tsx
import { ChatProvider } from "@openuidev/react-headless";

function App() {
  return (
    <ChatProvider apiUrl="/api/chat" threadApiUrl="/api/threads">
      <MyChat />
    </ChatProvider>
  );
}
```

See [Connecting to LLM](/docs/chat/chat-ui/connecting-to-llm) and [Conversation History](/docs/chat/chat-ui/conversation-history) for details on these props.

## Hooks

All hooks must be used inside a `ChatProvider`. Both support custom selectors for fine-grained re-renders (e.g. `useThread(s => s.messages)`).

### `useThread`

Access the current thread's state and actions:

```typescript
const {
  // State
  messages, // Message[] — the current conversation
  isRunning, // boolean — true while streaming a response
  isLoadingMessages, // boolean — true while loading thread history
  threadError, // Error | null — last error, if any

  // Actions
  processMessage, // (message: CreateMessage) => Promise<void> — send a message
  cancelMessage, // () => void — cancel the in-progress response
  appendMessages, // (...messages: Message[]) => void — add messages to the thread
  updateMessage, // (message: Message) => void — update a specific message
  setMessages, // (messages: Message[]) => void — replace all messages
  deleteMessage, // (messageId: string) => void — remove a message
} = useThread();
```

### `useThreadList`

Access the thread list state and actions:

```typescript
const {
  // State
  threads, // Thread[] — all conversations
  isLoadingThreads, // boolean — true while fetching the list
  threadListError, // Error | null — last error, if any
  selectedThreadId, // string | null — currently active thread
  hasMoreThreads, // boolean — true if more threads can be loaded (pagination)

  // Actions
  loadThreads, // () => void — load the thread list
  loadMoreThreads, // () => void — load next page of threads
  switchToNewThread, // () => void — start a new conversation
  createThread, // (firstMessage: UserMessage) => Promise<Thread>
  selectThread, // (threadId: string) => void — switch to a thread
  updateThread, // (thread: Thread) => void — rename, etc.
  deleteThread, // (threadId: string) => void
} = useThreadList();
```

## Example: full custom chat

A complete custom chat with a message list, input, and thread sidebar — no OpenUI UI components used:

```tsx
import { useState } from "react";
import { ChatProvider, useThread, useThreadList } from "@openuidev/react-headless";

function MessageList() {
  const { messages, isRunning } = useThread();

  return (
    <div className="messages">
      {messages.map((msg) => (
        <div key={msg.id} className={`message message-${msg.role}`}>
          {msg.content}
        </div>
      ))}
      {isRunning && <div className="typing-indicator">Thinking...</div>}
    </div>
  );
}

function ChatInput() {
  const { isRunning, processMessage, cancelMessage } = useThread();
  const [input, setInput] = useState("");

  return (
    <form
      onSubmit={(e) => {
        e.preventDefault();
        if (input.trim() && !isRunning) {
          processMessage({ role: "user", content: input });
          setInput("");
        }
      }}
    >
      <input
        value={input}
        onChange={(e) => setInput(e.target.value)}
        placeholder="Type a message..."
        disabled={isRunning}
      />
      {isRunning ? (
        <button type="button" onClick={cancelMessage}>
          Stop
        </button>
      ) : (
        <button type="submit">Send</button>
      )}
    </form>
  );
}

function ThreadSidebar() {
  const {
    threads,
    selectedThreadId,
    isLoadingThreads,
    selectThread,
    switchToNewThread,
    deleteThread,
  } = useThreadList();

  return (
    <aside className="sidebar">
      <button onClick={switchToNewThread}>+ New Chat</button>
      {isLoadingThreads && <div>Loading...</div>}
      {threads.map((t) => (
        <div key={t.id} className={t.id === selectedThreadId ? "active" : ""}>
          <button onClick={() => selectThread(t.id)}>{t.title}</button>
          <button onClick={() => deleteThread(t.id)}>×</button>
        </div>
      ))}
    </aside>
  );
}

function App() {
  return (
    <ChatProvider apiUrl="/api/chat" threadApiUrl="/api/threads">
      <div className="chat-app">
        <ThreadSidebar />
        <main>
          <MessageList />
          <ChatInput />
        </main>
      </div>
    </ChatProvider>
  );
}
```
