Use this package for LLM-driven email template generation with 44 email building blocks.

## Import

```ts
import { emailLibrary, emailPromptOptions } from "@openuidev/react-email";
```

## `emailLibrary`

Pre-configured `Library` instance (from `createLibrary()`) with all 44 email components registered. Root component is `EmailTemplate`.

```ts
const emailLibrary: Library;
```

Generate a system prompt:

```ts
const systemPrompt = emailLibrary.prompt(emailPromptOptions);
```

Render email output:

```tsx
import { Renderer } from "@openuidev/react-lang";

<Renderer response={response} library={emailLibrary} isStreaming={isStreaming} />;
```

## `emailPromptOptions`

Prompt configuration with 10 complete email template examples and 30+ rules for email design generation.

```ts
const emailPromptOptions: PromptOptions;
// { examples: emailExamples, additionalRules: emailAdditionalRules }
```

## Additional exports

These are also exported from `./library` for advanced customization:

```ts
import {
  emailLibrary,
  emailPromptOptions,
  emailComponentGroups,
  emailExamples,
  emailAdditionalRules,
} from "@openuidev/react-email";
```

| Export | Type | Description |
| :--- | :--- | :--- |
| `emailLibrary` | `Library` | Ready-to-use library with all 44 components |
| `emailPromptOptions` | `PromptOptions` | Combined examples + rules for `library.prompt()` |
| `emailComponentGroups` | `ComponentGroup[]` | 13 category groups with component names and usage notes |
| `emailExamples` | `string[]` | 10 complete email template examples in OpenUI Lang |
| `emailAdditionalRules` | `string[]` | 30+ rules for high-quality email generation |
