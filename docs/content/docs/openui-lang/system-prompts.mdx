---
title: System Prompts
description: Generate and customize prompt instructions from your OpenUI library.
---

`library.prompt(...)` generates the instruction text your model needs to output valid OpenUI Lang.

## Generate prompt

```ts
import { defaultLibrary, defaultPromptOptions } from "@openuidev/react-ui";

const systemPrompt = defaultLibrary.prompt(defaultPromptOptions);
```

## Prompt options

```ts
import type { PromptOptions } from "@openuidev/lang-react";

const options: PromptOptions = {
  preamble: "You are an assistant that outputs only OpenUI Lang.",
  additionalRules: ["Always produce actionable controls for forms.", "Prefer concise layouts."],
  examples: [`root = Stack([title])\ntitle = TextContent("Hello")`],
};

const prompt = defaultLibrary.prompt(options);
```

## What gets generated

The generated prompt includes:

- syntax rules
- component signatures (from your registered components)
- hoisting/streaming rules
- important output constraints
- your optional examples/rules

## Important behavior notes

- Root component requirement comes from the library `root` passed to `createLibrary`.
- Prompt signatures include component descriptions, argument names, and required/optional structure.

## Backend usage example

```ts
import OpenAI from "openai";
import { defaultLibrary, defaultPromptOptions } from "@openuidev/react-ui";

const client = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

export async function POST(req: Request) {
  const { messages } = await req.json();

  const completion = await client.chat.completions.create({
    model: "gpt-5.2",
    stream: true,
    messages: [
      { role: "system", content: defaultLibrary.prompt(defaultPromptOptions) },
      ...messages,
    ],
  });

  return new Response("stream here");
}
```
