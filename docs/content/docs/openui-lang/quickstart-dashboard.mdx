---
title: "Dashboard"
description: Bootstrap a data-driven dashboard app in under a minute.
---

## Create the app

```bash
npx @openuidev/cli@latest create --name my-dashboard --template dashboard
cd my-dashboard
```

## Add your API key and start

```bash
echo "OPENAI_API_KEY=sk-your-key-here" > .env
npm run dev
```

The dashboard template includes example tools, MCP endpoint, and a prompt config. Ask it to build a ticket tracker, analytics dashboard, or monitoring tool. The LLM generates the UI, and the runtime calls your tools directly.
